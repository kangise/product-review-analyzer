# 增强版消费者喜爱点分析 - 基于产品类型定制

## 产品类型特异性分析框架

### 如果 product_type = "360度全景相机"
**重点关注维度**：
- 视频质量（4K、防抖、低光表现）
- 易用性（一键拍摄、APP体验）
- 硬件耐用性（防水、抗摔）
- 创意功能（全景、VR、慢动作）

### 如果 product_type = "智能手表"  
**重点关注维度**：
- 健康监测（心率、睡眠、运动）
- 续航表现（电池寿命、充电速度）
- 智能交互（语音、触控、通知）
- 外观设计（表带、屏幕、重量）

### 如果 product_type = "无线耳机"
**重点关注维度**：
- 音质表现（低音、高音、降噪）
- 佩戴舒适度（入耳感、重量分布）
- 连接稳定性（蓝牙、延迟、断连）
- 续航与充电（单次使用、充电盒）
